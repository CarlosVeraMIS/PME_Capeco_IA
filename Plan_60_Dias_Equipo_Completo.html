IA como copiloto transversal

Claude no es una fase. Es el miembro 11 del equipo.

Cada rol tiene un caso de uso específico de Claude desde el Día 1. No hay un "sprint de IA". IA está embebida en cada pista desde el primer standup.

PM
📋
Gestión y reporte
Genera reportes de avance, gestiona el risk log, redacta comunicados a CAPECO y analiza blockers con contexto del proyecto.
DE
⚙️
Pipelines y agentes
Escribe y revisa pipelines Python/SQL, genera los agentes como scripts validados, documenta el schema Gold automáticamente.
FS
💻
Código y tests
Genera boilerplate de componentes, escribe tests unitarios desde el schema, revisa PRs y sugiere refactors de performance.
DA
📊
Análisis y demo
Analiza la BD en lenguaje natural, detecta anomalías, arma el storyline de la demo y genera los slides narrativos para CAPECO.
QA
🔍
Test cases
Genera casos de test desde el schema Gold, crea scripts de carga, documenta bugs con reproducción paso a paso.
